업비트 트레이딩 봇
=================

개요
----------
* 업비트 트레이딩봇을 개발하여 자동 매매를 진행해본다
* 매매/매수 알고리즘을 다른사용자들이 쉽게 수정할 수 있게 개발하여 자신만의 알고리즘을 공유할 수 있는 환경을 만든다.



개발환경
-------
* Pyhton 3.7
* Upbit API
* Visual Studio 2019
* PyQt5

------
* 개발내용은 기본적으로 노션에 작성합니다.
